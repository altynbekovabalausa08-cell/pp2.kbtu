# Practice 4 - Python Examples

This repository contains examples for Python practice, including:

1. **Date and Time** – Getting current date and time, creating specific dates, printing weekdays and months, and removing microseconds.  
2. **Generators** – Using `iter()` and `next()`, generator functions for even numbers and Fibonacci sequence, generator expressions, and square roots.  
3. **Math Module** – Calculating square roots, powers, ceiling and floor, converting degrees to radians, and generating random numbers.  
4. **JSON** – Converting Python objects to JSON and back, accessing JSON fields, and reading/writing JSON files.

These examples are designed for practice and learning Python basics.
