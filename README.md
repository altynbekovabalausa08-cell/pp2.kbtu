PhoneBook Database Project

This project is a simple PhoneBook application using PostgreSQL and Python. It demonstrates how to create and call functions and procedures, use PL/pgSQL basics such as variables, IF/ELSE, LOOP, and FOR, handle data validation, and implement pagination with LIMIT and OFFSET.

The project includes a function to search contacts by pattern, a procedure to insert or update a user, a procedure to insert many users with phone validation, a procedure to delete a user by name or phone, and a function to get contacts with pagination.

The Python program (main.py) provides a menu to interact with the database, load SQL files (functions.sql and procedures.sql), insert data from the console, search, paginate, and delete contacts.

Technologies used: Python, PostgreSQL, psycopg2, and PL/pgSQL.

This project demonstrates the integration of Python with PostgreSQL and basic database operations using functions and procedures.
