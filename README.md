This project contains practice tasks for learning Regular Expressions (RegEx) in Python  PRACTICE 5 . Different regex patterns are used to match, search, split, and replace text.
The exercises include working with character patterns, uppercase and lowercase sequences, and converting between snake_case and camelCase.
The project also includes a receipt parsing task where data such as product names, prices, date, and payment method are extracted from a text file.


This project also includes solutions for tasks such as:
matching patterns with a and b
finding lowercase words with underscores
detecting capitalized words
converting snake_case ↔ camelCase
splitting text by uppercase letters
replacing punctuation with :
