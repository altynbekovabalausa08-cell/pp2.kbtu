This project is a collection of simple scripts I wrote while learning the basics of Python.    What I practiced:
The Basics: How to write code that Python can read (Syntax).

Notes: How to add human-readable notes inside code (Comments).

Storing Info: Using "containers" for different types of data like text and numbers (Variables & Data Types).

Conversions: How to change one type of data into another, like turning a word into a number (Casting).

Text: Different ways to play with and display words (Strings).
